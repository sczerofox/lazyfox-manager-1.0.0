
							 LazyFox OSS Manager


----------------
--	  简介
----------------

是一个对象存储(OSS)工具。它基于 SpringBoot 架构编写的一个简单的文件存储服务工具，可以上传文件，删除文件，预览文件。

----------------
--   系统要求
----------------

JDK：1.8或更高版本（这是执行SpringBoot的基本要求 ）。

端口：4122

内存：没有最低要求。

磁盘：OSS-Manager-1.0.0 安装本身大约需要 35.0MB。

此外，OSS-Manager-1.0.0 repository 仓库将使用额外的磁盘空间。
你的本地仓库的大小将根据使用情况而变化，但至少需要500MB。 

操作系统： Windows：Windows 2000或更高版本。 
基于Unix的系统（Linux、Solaris和Mac OS X）和其他系统：没有最低要求。



------------------------------
-- 安装 OSS-Manager-1.0.0
------------------------------

将存档解压到你想要存储二进制文件的地方，例如： 基于Windows的操作系统 Windows unzip lazyfox-manager-1.x.y.zip

将 JAVA_HOEM8 环境变量设置好

双击 lazyfox-manager-1.0.0 下 或 lazyfox-oss-manager-1.0.0下
的 bin 中的 start.bat 即可成功启动服务
